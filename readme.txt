
Instruction to execute the code:

1) Type 'make' to compile the file.
2) make will execute the below code
	gcc backtrack.c -o backtrack.out -lm
	./backtrack.out knapsack.txt
	gcc brute.c -o brute.out -lm
	./brute.out knapsack.txt

Input files:
knapsack.txt - File containing the input data
backtrack.c  - File contains the knapsack solution using backtracking approach
brute.c      - File contains the knapsack solution using brute force approach

Below output files will be generated after the successfull execution:
1) backtrack_output.txt
2) brute_output.txt
